Abstract

Actopia is a project that aims to revolutionize film casting. It is a website designed to connect actors and actresses directly with film production companies, providing a secure and professional environment for casting. The website is built using HTML, PHP, and CSS, and features user and admin logins to manage user details. Actopia offers various pages, each dedicated to a specific type of information, and aims to eliminate fake casting calls by facilitating direct communication between actors and production houses.

Introduction

Actopia is a web-based casting call management system that streamlines the casting process for production companies and applicants. The system features user and admin login functionality, allowing users to apply for casting calls and admins to manage and review applications.

  • User profiles with editable information
  • Application submission and management
  • Admin review and feedback system
  • Casting call posting and management

This project aims to improve the efficiency and organization of the casting process, providing a user-friendly platform for both applicants and production companies.
